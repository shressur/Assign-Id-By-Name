Problem: A hospital logging system assign id numbers to patient based on their names. Ids take the form of 2 integers seperated by a decimal, so for example the first patient in a log would be "1.1". When patient share a last name the 2nd number in the id is increamented, so "1.1" now becomes "1.2", and if a subsequent patinet had a new last name the first number is increamented i.e. now the the third patient's id would be "2.1".
Given this behavior, create a method that assigns id numbers given a list of patient.

Test cases:

            
                A: Allen 	-> 1.1
                A: Baldwin      -> 2.1
                B: Baldwin      -> 2.2
                A: Carter 	-> 3.1
                C: Baldwin      -> 2.3
                B: Allen 	-> 1.2
                A: Davis 	-> 4.1
                C: Allen 	-> 1.3

Code:

            
                arr_uniqueNames = [];
                arr_allNames = [];
                function assignId(name){
                    //keep track of all the patients
                    arr_allNames.push(name);
                    if(!arr_uniqueNames.includes(name)){
                        //keep track of unique names based on the array index
                        arr_uniqueNames.push(name);
                    }
                    //[ ].indexOf() => 0 based
                    //"arr_allNames" is never empty so "[ ].filter().length" > 0 (counting duplicates)
                    return (arr_uniqueNames.indexOf(name) + 1) + "." + arr_allNames.filter(e => e == name).length;
                }
